# Sistema de Login com API "Reqres" 

#### Nesse projeto simulei o login de um usuário, gerenciando suas informações com o Context do React e LocalStoreAge, para isso utilizei a API Reqres. 


- Principais assuntos:
  - Autenticação;
  - Context API (Hook); 
  - Gerenciamento de estado;
  - Consumo de API;
  - LocalStoreAge;
